
# Rupayana - Digital Payment Website (Demo)

This project contains a full-stack demo for:
- Sign up / Login
- Forgot Password (email link)
- Account Management
- Fund Transfer
- Bill Payment
- Transaction History
- Admin Controls

## How to Run

### Backend
1. `cd backend`
2. `npm install`
3. Create `.env` from `.env.example`
4. `node server.js`

### Frontend
Just open `frontend/index.html` in your browser.

This project is meant as a demo template for learning purposes.
